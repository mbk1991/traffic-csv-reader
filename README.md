# traffic_csv_reader
:traffic csv 파일을 1분에 1회 읽어서 DB에 적재하는 프로그램
<br>라우터 -> nfcapd -> nfdump -> (csv파일)

<li>스레드풀 사이즈 설정</li>
<li>커넥션풀 사이즈 설정</li>
</ol>
